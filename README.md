# Agente de noticias

Agente automatizado que recopila noticias de economía y política desde Infobae, La Nación y Clarín, las agrupa y resume con IA, y publica un resumen diario junto con indicadores financieros del día. Se ejecuta solo varias veces al día mediante GitHub Actions.

Ver el resumen publicado en: https://msmassolo.github.io/Resumen_de_noticias/
